# Devices-inventory-website
```
Let's say you are an IT professional and in your workplace, there is a broken printer or computer that you want to repair.
So, through this site, you can search for this printer or computer based on (name, barcode, model..) to find its location and
go to fix it. vice versa. in addition, you can add/edit printers or PC.
tech used : python(django) , sqlite , django templates.

```
